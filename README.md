## Flask Portfolio Website

This project is a personal portfolio website built using Flask, showcasing how backend frameworks can be used to serve dynamic HTML templates and static assets.

The application uses Flask to render an HTML template and serve CSS, images, and JavaScript files from the static directory. It demonstrates how to structure a web project using templates and static resources in a clean and maintainable way.

This project was built as part of the #90DaysOfCode challenge to practice backend integration with frontend templates using Python and Flask.

---

## Technologies Used

Python  
Flask  
HTML5

## Key Concepts Demonstrated

Rendering HTML templates using Flask

Using Jinja templating with render_template

Serving static files such as CSS, images, and JavaScript

Project structuring with templates and static folders

Building a responsive portfolio layout

Integrating frontend UI with a backend framework

---

## Project Structure

```
project/
│
├── server.py
├── templates/
│   └── index.html
├── static/
│   ├── css files
│   ├── images
│   └── javascript files
```

---

## Application Workflow

1. Initialize a Flask application
2. Define a route for the homepage
3. Render the index.html template using render_template
4. Load static assets such as CSS, images, and scripts
5. Display a responsive portfolio website in the browser

```

```


Then open your browser and go to

```

```
